                Step-by-Step Installation Guides
            

            
            

                
                
                
                
                
                
            


            
            

                
Option 1: Add via Custom Liquid (Fastest, works on any theme)

                
    
                    
  • Open your Shopify Admin, go to Online Store > Themes, and click Customize on your active theme.
    • 
                    
  • In the left-hand sidebar, click Add Section (or Add Block inside an existing section) and select Custom Liquid.
    • 
                    
  • Copy the 1-Line Script snippet from the configuration box above.
    • 
                    
  • Paste the snippet directly into the **Custom Liquid** editor box on the right sidebar.
    • 
                    
  • Click Save. The widget will load dynamically and automatically fit your layout.
    • 
                

                
                
Option 2: Install as a Custom Theme Section (Native Drag & Drop)

                
    
                    
  • In Shopify Admin, go to Online Store > Themes, click the three dots ... next to your theme, and select Edit Code.
    • 
                    
  • Under the Sections folder on the left side, click Add a new section, name it today-in-history, and click Create.
    • 
                    
  • Open the today-in-history.liquid file from your local workspace (located in shopify_theme/sections/) and copy all of its code.
    • 
                    
  • Paste the code into your Shopify editor, replacing any default template code, and click Save.
    • 
                    
  • Return to your Theme Customizer. You can now add the section Today in American History anywhere as a native, drag-and-drop block with built-in color controls.
    •

Option 1: Inject Widget in Existing Page

                
    
                    
  • Add a custom HTML block, custom liquid section, or custom script container to your website builder.
    • 
                    
  • Copy the 1-Line Script snippet and paste it into the editor. It will render in-place.
    • 
                

                
Option 2: Add as a Standalone New Page (Easiest Link)

                
    
                    
  • In your website builder menu editor, add a new link item called "Liberty Ledger".
    • 
                    
  • Copy the Direct Page Link URL from the snippet output box above.
    • 
                    
  • Set the new menu item to link directly to that URL. Clicking it will load your custom white-labeled page.
