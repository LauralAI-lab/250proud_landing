                Step-by-Step Installation Guides
            

            
            

                
                
                
                
                
            


            
            

                
    
                    
  • Search for Today in History in the Shopify App Store and click Install.
    • 
                    
  • Open your Shopify admin dashboard, click Online Store > Themes, and select Customize on your active theme.
    • 
                    
  • Navigate to the template (e.g., Homepage or custom landing page) where you want the widget to appear.
    • 
                    
  • In the left sidebar, click Add Section or Add Block, and select Today in History widget from the Apps tab.
    • 
                    
  • Customize the accent colors, theme (light/dark), and CTA links directly inside the Shopify block panel, then click Save.
    • 
                

            


            
            

                
    
                    
  • In your Wix Editor, click the Add Elements (+) button on the left sidebar.
    • 
                    
  • Navigate to Embed Code and select HTML Embed.
    • 
                    
  • Click Enter Code above the placeholder box that appears on your screen.
    • 
                    
  • Select Code, paste the 1-Line Script snippet (copied from the builder above), and click Update.
    • 
                    
  • Drag the borders of the embed box to size it cleanly on your page (recommended width: 100%, height: 550px).
    • 
                

            


            
            

                
    
                    
  • Open your GoDaddy Website Builder editor.
    • 
                    
  • Scroll to the page section where you want the widget to appear, and click Add Section.
    • 
                    
  • Search for HTML or Custom Code in the section category list.
    • 
                    
  • Paste the HTML IFrame snippet or 1-Line Script snippet directly into the Custom Code text area.
    • 
                    
  • Set the section height to auto or set iframe height to 600px, click Done, and publish.
    • 
                

            


            
            

                
    
                    
  • Edit the page on your Squarespace site, and click Add Block in the desired section.
    • 
                    
  • Select Code or Embed from the block options menu.
    • 
                    
  • Click the edit pencil icon on the block, choose HTML as the format, and paste the 1-Line Script snippet.
    • 
                    
  • Make sure the 'Display Source Code' option is toggled off.
    • 
                    
  • Adjust the block dimensions to stretch across your column space cleanly.
    • 
                

            


            
            

                
Option 1: Inject Widget in Existing Page

                
    
                    
  • Add a custom HTML block, custom liquid section, or custom script container to your website builder.
    • 
                    
  • Copy the 1-Line Script snippet and paste it into the editor. It will render in-place.
    • 
                

                
Option 2: Add as a Standalone New Page (Easiest Link)

                
    
                    
  • In your website builder menu editor, add a new link item called "Today In History".
    • 
                    
  • Copy the Direct Page Link URL from the snippet output box above.
    • 
                    
  • Set the new menu item to link directly to that URL. Clicking it will load your custom white-labeled page.
